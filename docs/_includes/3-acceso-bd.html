## Mecanismos de acceso a MySQL/MariaDB - Php proporciona tres formas de conectar nuestra aplicación a un motor de bases de datos MySQL/MariaDB: 1. Usando métodos de la librería mysqli 2. Usando clases proporcionadas por la librería mysqli 3. Usando Php Data Object (PDO) - En la actualidad, la elección más sensata es usar PDO, ya que es una conexión común a varios motores de bases de datos. - De esta forma podemos desarrollar una aplicación que pueda funcionar usando Oracle o Postgre, por ejemplo.
